# cadastro-de-usuario-php
Cadastro de usuário simples com o objetivo de entender melhor as tecnologias: PHP, JavaScript, MYSQL, HTML E CSS.


 Sistema de Cadastro de Usuários

 Projeto simples desenvolvido para praticar desenvolvimento web com PHP, MySQL, HTML e JavaScript.

 Funcionalidades:
- Cadastro de usuários (nome e email)
- Validação de campos com JavaScript
- Armazenamento de dados no MySQL
- Listagem de usuários cadastrados


 Tecnologias utilizadas:
- PHP
- MySQL
- HTML
- JavaScript
